API Documentation

Estoquim API

Esta é a página única de documentação da API. Todos os recursos do projeto são expostos apenas via endpoints REST, não há interface web adicional além desta página.

Como usar

Use requisições HTTP com o método indicado em cada endpoint. Envie JSON no corpo da requisição quando o endpoint aceitar dados. Veja abaixo a lista de endpoints disponíveis.

Endpoints disponíveis

  • Registrar usuário POST /userprofile/register/

    Cria um novo usuário e perfil associado.

    Exemplo de body JSON:

    {
  "username": "usuario",
  "password": "senha123",
  "email": "usuario@exemplo.com",
  "birthdate": "1990-01-01"
}
  • Login de usuário POST /userprofile/login/

    Autentica credenciais e cria sessão para o usuário.

    Exemplo de body JSON:

    {
  "username": "usuario",
  "password": "senha123"
}
  • Ver perfil do usuário POST /userprofile/view/

    Retorna os dados do usuário autenticado. Requer sessão ativa.

  • Listar usuários POST /userprofile/list/

    Retorna a lista de perfis de usuários. Requer usuário staff autenticado.

Observações importantes

Esta página é apenas para documentação.
Todos os endpoints aceitam JSON e retornam JSON quando suportado.
Requisições GET não são usadas nos endpoints listados, apenas POST.
